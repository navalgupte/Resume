Transient Noise Bounds for PDN/PG using Vectorless Verification (Paper)
Power grid (PG) simulation is computationally demanding due to increased grid sizes
Vectorless approach estimates worst-case noises without detailed load current evaluations
Transient constraints are utilized to capture transitory circuit behavior
Abstract grid model is utilized to accelerate computations
Ways to mitigate storage and computational requirements on processing resources are proposed
Computations can be deployed on economical Cloud computing resources
Recommended solution is parallelizable thereby reducing overall verification time
